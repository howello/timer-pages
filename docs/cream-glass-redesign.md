# 奶油毛玻璃新拟态改版设计稿

## 目标

将 `password.html` 和 `index.html` 从基础毛玻璃界面升级为更大胆的时间主题作品集式体验，同时保持倒计时工具的清晰可用。页面应呈现新拟态、毛玻璃、奶油风，并通过更强首屏仪式感让应用更有记忆点。

## 视觉方向

- 底色：奶油白、燕麦米、暖灰作为主背景。
- 点缀：珊瑚红、鼠尾草绿、淡天蓝、柔紫用于按钮、卡片状态和时间轨迹。
- 质感：半透明玻璃层、内凹新拟态输入框、柔和投影、细腻纸纹和时间刻度线。
- 形状：保持 8px 圆角，避免过度圆润；装饰使用轨道、刻度、面板和光泽，而不是漂浮色球。
- 字体：系统中文字体，标题加粗但不拥挤，数字使用 tabular-nums 保持倒计时稳定。

## 首页结构

1. 顶部悬浮操作栏
   - 保留新增、同步、登出按钮。
   - 采用玻璃胶囊布局，滚动后浮现，保持现有交互逻辑。

2. 首屏时间舞台
   - 左侧为品牌标题、当前时间、应用状态。
   - 右侧为时间轨道装饰和一个仪式感强的玻璃面板。
   - 固定倒计时卡片仍由 `.fixed-card-stage` 渲染，但视觉上成为首屏主角。

3. 事件清单
   - 作为下方工作台出现。
   - 列表卡片更轻、更整齐，保留拖拽、置顶、编辑、删除。

4. 新增/编辑弹窗
   - 保留字段和 DOM 钩子。
   - 改为更精致的玻璃工作面板，输入框使用内凹新拟态。

## 密码页结构

1. 入口舞台
   - 全屏奶油背景，叠加时间刻度与纸纹。
   - 一侧展示品牌和简短状态，另一侧是登录卡。

2. 登录卡
   - 保留 `#password-form`、`#password-input`、`#toggle-password`、`#status-message`。
   - 密码输入框和提交按钮突出触感。
   - 错误和成功状态沿用现有 JS 类名。

## 交互原则

- 不改动 API、鉴权、数据存储和现有 JS 行为。
- 保留所有现有 ID：`add-event-btn`、`sync-btn`、`logout-btn`、`fixed-card-stage`、`event-list`、`modal-backdrop`、`event-form`、`password-form`。
- 响应式优先：桌面双区舞台，移动端单列，操作按钮不挤压标题。
- 动效轻量：高光流动、hover 抬升、滚动浮现；尊重 `prefers-reduced-motion`。
